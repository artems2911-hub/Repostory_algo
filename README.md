# Memory card


```
Memory card - приложение для проверки своих знаний

После запуска нужно пройти тест в окне с названием Memory card и после этого посмотреть результат в другом окне
При желании можно добавить вопросы в этот тест, используя код этого приложения

Плюсы данного приложения:
1. Простой интерфейс
2. Возможность редактировать вопросы и добавлять новые, используя код
3. Приложение можно использовать для проверки знаний учеников
4. Оценка правильности ответов уже встроена, и она в процентах ('%')

Чтобы создать исполняемый файл данного проекта необходимо выполнить следующие шаги:
1. Скопировать код проекта в приложение VSC (В PyCharm могут быть другие действия)
2. Проверить есть ли pyinstaller при помощи оператора pip list (вводиться в консоль)
3. Если его нет, установить его при помощи оператора pip install pyinstaller, вводится так же в консоль (при наличии pyinstaller пропустить шаг)
4. В консоль ввести оператор pyinstaller --onefile <имя данного проекта>.py (как записан у Вас)
5. После того как в консоль выведется сообщение что операция выполнена, зайти в папку проекта
(комбинация клавишь Shift+Alt+R. Если не сработало, то: ЛКМ по кнопке dist и нажать на reveal in file explorer)
6. Вынести из dist испольняемый файл на рабочий стол или в необходимую папку
Приложение готово к работе
```
